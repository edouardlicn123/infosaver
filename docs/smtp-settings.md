# SMTP设置模块

## 一、模块概述

管理第三方SMTP服务器配置，支持多账号切换和连接测试。

## 二、数据库设计

### smtp_settings 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| name | TEXT | 配置名称 |
| smtp_host | TEXT | SMTP服务器地址 |
| smtp_port | INTEGER | 端口号 |
| smtp_user | TEXT | 用户名 |
| smtp_password | TEXT | 密码/授权码 |
| use_tls | BOOLEAN | 是否启用TLS |
| sender_name | TEXT | 发件人显示名 |
| sender_email | TEXT | 发件人邮箱 |
| is_default | BOOLEAN | 默认配置标记 |
| created_at | DATETIME | 创建时间 |

## 四、路由设计

| 路由 | 方法 | 功能 |
|------|------|------|
| `/smtp` | GET | 显示配置列表 |
| `/smtp/add` | GET | 显示添加表单 |
| `/smtp/add` | POST | 处理添加请求 |
| `/smtp/<id>/edit` | GET | 显示编辑表单 |
| `/smtp/<id>/edit` | POST | 处理编辑请求 |
| `/smtp/<id>/delete` | POST | 删除配置 |
| `/smtp/<id>/test` | POST | 测试连接 |

## 五、功能逻辑

### 5.1 默认配置

- 只有一个配置可设为默认
- 设为默认时，自动取消其他配置的默认标记
- 发信时使用默认配置

### 5.2 测试连接

```python
def test_smtp_connection(config):
    try:
        with smtplib.SMTP(config.smtp_host, config.smtp_port) as server:
            server.starttls()
            server.login(config.smtp_user, config.smtp_password)
        return True, "连接成功"
    except Exception as e:
        return False, str(e)
```

### 5.3 常见SMTP配置

| 邮箱服务 | SMTP服务器 | 端口 | TLS |
|----------|------------|------|-----|
| QQ邮箱 | smtp.qq.com | 587 | 是 |
| 163邮箱 | smtp.163.com | 465/587 | 是 |
| Gmail | smtp.gmail.com | 587 | 是 |
| 企业邮箱 | smtp.company.com | 465/587 | 是 |

### 5.4 密码存储

- 密码以明文存储在数据库
- 建议使用专用授权码而非登录密码
- 数据库文件应妥善保管

## 六、验证规则

| 字段 | 验证规则 |
|------|----------|
| 配置名称 | 必填，最大50字符 |
| SMTP服务器 | 必填，合法域名或IP |
| 端口 | 必填，1-65535整数 |
| 用户名 | 必填，有效邮箱格式 |
| 密码 | 必填，非空 |
| 发件人邮箱 | 必填，有效邮箱格式 |
