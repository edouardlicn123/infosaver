# 发信日志模块

## 一、模块概述

记录每次批量发送的历史，支持查询和详情查看。

## 二、数据库设计

### email_logs 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| content_id | INTEGER | 使用的模板ID |
| smtp_config_id | INTEGER | 使用的SMTP配置ID |
| subject | TEXT | 实际发送的邮件主题 |
| recipient_count | INTEGER | 收件人数量 |
| success_count | INTEGER | 成功数量 |
| fail_count | INTEGER | 失败数量 |
| status | TEXT | 状态（success/partial/failed） |
| error_message | TEXT | 错误信息汇总 |
| sent_at | DATETIME | 发送时间 |

## 四、路由设计

| 路由 | 方法 | 功能 |
|------|------|------|
| `/logs` | GET | 显示日志列表 |
| `/logs/<id>` | GET | 显示日志详情 |

## 五、状态说明

| 状态值 | 显示 | 说明 |
|--------|------|------|
| success | ✓ 成功 | 全部发送成功 |
| partial | ⚠ 部分成功 | 部分成功，部分失败 |
| failed | ✗ 失败 | 全部失败 |

## 六、显示要点

### 6.1 列表页

- 按发送时间倒序排列（最新在上）
- 支持按状态筛选
- 支持按时间范围筛选
- 支持按邮件主题搜索
- 显示成功/失败比例

### 6.2 详情页

- 显示发送统计（总数、成功数、失败数）
- 显示使用的模板和SMTP配置
- 失败时显示失败原因
- 显示本次发送的所有收件人

### 6.3 与客户管理联动

日志详情中的客户列表支持点击跳转到客户详情页，方便查看客户完整信息。
