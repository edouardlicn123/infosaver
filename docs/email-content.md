# 发信内容管理模块

## 一、模块概述

创建和管理邮件模板，支持富文本编辑和变量占位符。

## 二、数据库设计

### email_contents 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| title | TEXT | 邮件标题 |
| body | TEXT | 邮件内容（HTML格式） |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

## 四、路由设计

| 路由 | 方法 | 功能 |
|------|------|------|
| `/content` | GET | 显示模板列表 |
| `/content/add` | GET | 显示创建表单 |
| `/content/add` | POST | 处理创建请求 |
| `/content/<id>/edit` | GET | 显示编辑表单 |
| `/content/<id>/edit` | POST | 处理编辑请求 |
| `/content/<id>/delete` | POST | 删除模板 |

## 五、富文本编辑器

### 5.1 技术选型

使用 Quill.js（CDN引入），轻量级、功能完善。

| 功能 | 说明 |
|------|------|

## 六、变量替换机制

### 6.1 支持的变量

| 变量 | 对应字段 | 示例 |
|------|----------|------|
| `{name}` | customers.name | 张三 |
| `{email}` | customers.email1 | zhangsan@example.com |
| `{company}` | customers.company | ABC公司 |
| `{position}` | customers.position | 产品经理 |

### 6.2 替换逻辑

```python
def replace_variables(template, customer):
    replacements = {
        '{name}': customer.name or '',
        '{email}': customer.email1 or '',
        '{company}': customer.company or '',
        '{position}': customer.position or ''
    }
    content = template.title  # 标题也替换
    body = template.body
    
    for key, value in replacements.items():
        content = content.replace(key, value)
        body = body.replace(key, value)
    
    return content, body
```

### 6.3 使用示例

**原始模板：**
```

```

```
我们很高兴宣布...
```
