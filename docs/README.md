# 邮件系统扩展 - 总体设计文档

## 一、扩展概述

在现有客户信息管理系统基础上，增加邮件发送功能模块，实现批量、自动化发送邮件。

### 1.1 新增模块

| 模块 | 说明 | 优先级 |
|------|------|--------|
| SMTP设置 | 配置第三方SMTP服务器 | P0 |
| 发信内容管理 | 创建和管理邮件模板 | P0 |
| 发信功能 | 选择模板和收件人，执行发送 | P0 |
| 发信日志 | 记录发送历史，便于追溯 | P1 |

## 二、数据库设计

### 2.1 新增数据表

#### SMTP配置表 (smtp_settings)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| name | TEXT | 配置名称（如"公司邮箱"） |
| smtp_host | TEXT | SMTP服务器地址 |
| smtp_port | INTEGER | 端口号（默认587） |
| smtp_user | TEXT | 用户名/邮箱 |
| smtp_password | TEXT | 密码或授权码 |
| use_tls | BOOLEAN | 是否使用TLS（默认True） |
| sender_name | TEXT | 发件人显示名称 |
| sender_email | TEXT | 发件人邮箱地址 |
| is_default | BOOLEAN | 是否为默认配置 |
| created_at | DATETIME | 创建时间 |

#### 邮件模板表 (email_contents)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| title | TEXT | 邮件标题 |
| body | TEXT | 邮件内容（HTML格式） |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

#### 发信日志表 (email_logs)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| content_id | INTEGER | 使用的模板ID |
| smtp_config_id | INTEGER | 使用的SMTP配置ID |
| subject | TEXT | 实际发送的邮件主题 |
| recipient_count | INTEGER | 收件人数量 |
| status | TEXT | 状态（success/failed） |
| error_message | TEXT | 错误信息（如有） |
| sent_at | DATETIME | 发送时间 |

### 2.2 修改现有表

#### 客户表 (customers) - 新增字段

| 字段 | 类型 | 说明 |
|------|------|------|
| last_email_sent_at | DATETIME | 最后发送邮件时间（可NULL） |

## 四、变量替换机制

邮件模板支持动态变量，发送时自动替换。

### 4.1 支持的变量

| 变量 | 对应字段 | 说明 |
|------|----------|------|
| `{name}` | customers.name | 客户姓名 |
| `{email}` | customers.email1 | 客户邮箱 |
| `{company}` | customers.company | 客户公司 |
| `{position}` | customers.position | 客户职位 |

### 4.2 使用示例

```
标题: 尊敬的 {name}，您好！
正文: 
尊敬的 {name}：

感谢您对 {company} 的关注...
```

发送时系统自动将 `{name}` 替换为实际客户姓名。

| `/content/<id>/delete` | POST | 删除模板 | 发信内容 |
| `/sender` | GET | 发信页面 | 发信功能 |
| `/sender/confirm` | POST | 确认发送 | 发信功能 |

| `/logs` | GET | 日志列表 | 发信日志 |
