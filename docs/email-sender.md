# 发信功能模块

## 一、模块概述

选择邮件模板和收件人，执行批量发送。发送过程可配置间隔时间。

## 二、发送流程

```
┌─────────────────┐
│   选择模板      │  ← 从邮件模板列表选择
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   选择收件人    │  ← 从客户列表筛选
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   发送设置      │  ← 设置间隔时间、选择SMTP
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   确认预览      │  ← 显示收件人列表确认
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   执行发送      │  ← 遍历发送、更新日志
└─────────────────┘
```

## 四、路由设计

| 路由 | 方法 | 功能 |
|------|------|------|
| `/sender` | GET | 显示发信页面 |
| `/sender/confirm` | POST | 显示确认页面 |
| `/sender/send` | POST | 执行发送 |
| `/sender/progress` | GET | 获取发送进度（AJAX） |

## 五、核心功能

### 5.1 收件人筛选

- 搜索：按姓名/邮箱/公司搜索
- 筛选：按国籍、按公司
- 分页：支持选择多页
- 计数：实时显示已选择人数

### 5.2 发送间隔

```python
# 发送配置
SEND_INTERVAL = 30  # 默认30秒

def send_email_batch(recipients, email_content, smtp_config, interval=30):
    success_count = 0
    fail_count = 0
    
    for customer in recipients:
        # 替换变量
        subject, body = replace_variables(email_content, customer)
        
        try:
            send_email(
                to_email=customer.email1,
                subject=subject,
                body=body,
                smtp_config=smtp_config
            )
            # 更新客户最后发送时间
            customer.last_email_sent_at = datetime.now()
            db.session.commit()
            success_count += 1
        except Exception as e:
            fail_count += 1
        
        # 间隔等待
        if customer != recipients[-1]:  # 最后一个不等待
            time.sleep(interval)
    
    # 记录日志
    log = EmailLog(
        content_id=email_content.id,
        smtp_config_id=smtp_config.id,
        subject=email_content.title,
        recipient_count=len(recipients),
        success_count=success_count,
        fail_count=fail_count,
        status='success' if fail_count == 0 else 'partial',
        sent_at=datetime.now()
    )
    db.session.add(log)
    db.session.commit()
```

### 5.3 SMTP发送

```python
from flask_mail import Mail, Message

mail = Mail(app)

def send_email(to_email, subject, body, smtp_config):
    msg = Message(
        subject=subject,
        recipients=[to_email],
        html=body,
        sender=(smtp_config.sender_name, smtp_config.sender_email)
    )
    
    with smtplib.SMTP(smtp_config.smtp_host, smtp_config.smtp_port) as server:
        if smtp_config.use_tls:
            server.starttls()
        server.login(smtp_config.smtp_user, smtp_config.smtp_password)
        server.send_message(msg)
```

## 六、客户列表排序

发送后，客户表的 `last_email_sent_at` 字段会被更新，便于后续按最新发信时间排序：

```sql
SELECT * FROM customers 
ORDER BY last_email_sent_at DESC NULLS LAST;
```

## 七、安全与限制

| 措施 | 说明 |
|------|------|
| 发送前确认 | 必须经过确认页面，不可直接发送 |
| 间隔控制 | 防止被SMTP服务器识别为垃圾邮件 |
| 错误处理 | 单个失败不影响其他发送 |
| 日志记录 | 所有发送操作记录到日志表 |
